Export data

Download a fresh MasterData_*.xlsx with three sheets:

  • MaterialMaster — every raw material, with current stock and ZAR value of stock on hand.
  • ProductSpec — the full BOM. One row per (sellable SKU × material) line with quantity.
  • ChangeLog — every recorded change to master/spec data, with timestamp and user.

The .xlsx is generated on demand. Open it, edit it, share it — the file is yours. The app does not write to it after download.

⬇ Download MasterData.xlsx

Import data

Upload a MasterData_*.xlsx to apply changes back to the database. Materials are matched by SKU; BOM lines by (PV SKU, Material SKU).

Default behaviour: additive — rows in the file are upserted, anything in the DB but not in the file is left alone. Tick Prune if you want missing rows to be removed too.
